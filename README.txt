HSK Master Pro - bản web app chạy trực tiếp

Cách chạy nhanh:
1. Mở file index.html trong trình duyệt.
2. Hoặc dùng 1 local server tĩnh nếu muốn thử như app web/PWA.

Gồm:
- Intro loading screen
- Menu 3 gạch
- 4 module: Từ vựng, Nói, Đọc, Viết
- Audio nói dùng Web Speech API (iPhone/Safari sẽ dùng voice hệ thống nếu có)

Lưu ý:
- Dữ liệu học liệu hiện là bộ starter content để chạy app ngay.
- Cấu trúc đã sẵn để mở rộng thành bộ HSK đầy đủ.
